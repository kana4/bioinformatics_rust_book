Bioinformatics in Rust

Biological File Types

Biological data is packaged in file types for targeted purposes. If a variant analysis was done, we might get back a variant call file (VCF), which will annotate variants from a genome reference. Sequencing Alignment Mapping (SAM) files are usually a lot larger and contain all the sequencing data aligned to our genome reference. Fasta files (not really short for anything) are the simplest of all; they only contain sequences and descriptions of the sequences.
